RUNNING ANACIN-X

In this chapter we will provide step by step instructions on installing the ANACIN-X software in your system. We will aslo provde the necessary commands and demonestrate the process.

Pulling Existing Repository from Docker Hub 

In this section we provide commands and show how to pull the ANACIN-X software form Docker Hub. 

  $ mkdir results
  $ module load apptainer
  $ apptainer pull --disable-cache oras://docker.io/yejashi/anacinx_prod:ppc64le
  $ apptainer build --sandbox dir_anacinx anacinx_prod_ppc64le.sif

Getting Jupyter Notebook

The jupyter notebook file and other supporting python scripts are available in GCLAB GitHub repository.In order to clone the reoository, you can follow the following steps:

  • Navigate to the directory where you want to clone the repository.
  • Run following command: 
    •  $ git clone https://github.com/Yejashi/anacin_jupyter.git

Configuring Jupyter Notebook

In order to configure jypyter notebook server and your brwoser. You will need to enter following commands:

  $ ssh -L 8888:localhost:8888 username@systemdomainname
  $ module load apptainer
  $ spack load openmpi
  $ jypyter notebook --no-browser --port-8888
